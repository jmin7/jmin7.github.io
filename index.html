CITIES

A full-stack Ruby on Rails application

Click here for CITIES blog

A travel community for users to share city-specific tips about their favorite locations around the world.
Users are able to link to specific city from homepage and view a list of posts on the specitic city page.

User cannot save invalid data to the database according to the validation rules
- Post's title must be between 1 and 200 characters.
- Post's body/content must not be empty.

HANGMAN

Web application based on HTML & CSS

Click here for Hangman Game

This is a hangman game with 7 chances to guess the correct word.
Once the word for the game is chosen, each letter is shown in a blank.
Once a letter is clicked, the button becomes unclickable so the user can't guess the same letter twice.
The game shows score and the score can be cleared by the 'New Player' button.

Languages: HTML5, Javascript, DOM, JQuery
Design: CSS3, Google Fonts
